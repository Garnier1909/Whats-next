チームで作業する際に、タスクを共有できるアプリを作りました。
ログイン後に表示されるホーム画面でタスクの確認、右上のナビゲーションからログアウトやメンバーの管理が行えます。

タスクの重要度や進捗を記号に変換して表示することで、文字で見るより分かりやすくなる様にしました。
